Window object Summary

using the keyword this refers to the window object. You can also access it using window. The window object contains information about the browser and some functions come from there such as alert() or isNaN(). You can manipulate the browser in some ways but this is mostly discouraged. There are few cases it is apropriate to manipulate the users browser so be careful not to annoy your users!

HTML5 API Summary

The data- attribute can be used to store custom data with the tags in html. Elements have a dataset property that can be used to access this data. You can also use document.querySelector.HTML5 has many apis that provide additional features. There is an event related to localStorage called 'storage'. You can use web workers to carry out processes in the background that could hold your page up. Websockets provide two-way communication with the server. The connection is kept open. !NOTE! Storing in JSON object - is in memory. As soon as you close page/browser it is gone. Local storage - is a persistent storage. You may save you JSON object as stringified in it. And if you close your browser you may then get it back. This is the testTag that stores a data-test attribute! !NOTE! The video tag works best for files you have downloaded rather than from the browser.
